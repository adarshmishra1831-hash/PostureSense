PostureSense: Smart Exercise Posture Detection
PostureSense is an AI-powered system that analyzes human exercise posture from uploaded videos.
It uses MediaPipe to extract body landmarks, applies a machine learning model to classify posture, and provides analytics through an R-based dashboard.
